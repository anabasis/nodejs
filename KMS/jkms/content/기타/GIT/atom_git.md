## git-plus

- Git Path : Git프로그램 위치
- Command Palette[Control + Shift + P] --> Git 명령어 실행

## git-clone

- Git에 있는 소스 Clone해 옴

- Command Palette[Control + Shift + P] --> Git Clone : Clone
- github 주소<https://github.com/anabasis/nodejs.git> 입력
