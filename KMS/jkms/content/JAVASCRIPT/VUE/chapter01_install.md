## NPM

```bash
# vue-cli 설치
$ npm install -g vue-cli

# "webpack" 템플릿을 이용해서 새 프로젝트 생성
$ vue init webpack vue-project

# 의존성을 설치하고 실행하세요!
$ cd vue-project

$ npm install
$ npm run dev
```
